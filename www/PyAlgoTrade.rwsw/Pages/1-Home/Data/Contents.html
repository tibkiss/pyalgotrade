PyAlgoTrade is a Python library for backtesting stock trading strategies.

Main features:

  • Support for Market, Limit, Stop and StopLimit orders.
  • Support for Yahoo! Finance CSV files.
  • Support for NinjaTrader CSV files.
  • Sharpe ratio analysis.
  • Max. drawdown and max. drawdown duration analysis.
  • Very easy to scale horizontally, that is, using one or more computers to backtest a strategy. You can even take advantage of Google's cloud computing infrastructure to run your strategies inside Google App Engine.
  • TA-Lib integration.
